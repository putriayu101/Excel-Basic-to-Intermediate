# 📊 Basic Excel 
This self learning folder contains **basic Excel learning materials** focused on building a strong foundation in formulas and data formatting.


## 🧩 Excel Topics Covered
### 1. Basic Functions
* SUM
* AVERAGE
* COUNT / COUNTA
* MIN / MAX

### 2. Date & Time Formulas
* TODAY
* NOW
* DATE
* MONTH / YEAR
* DATEDIF

### 3. Concatenate
* CONCAT
* CONCATENATE

### 4. IF Function
* IF
* Nested IF
* IF with AND / OR

### 5. Text Functions
* LEFT / RIGHT / MID
* LEN
* TRIM
* UPPER / LOWER / PROPER
* TEXTJOIN

### 6. Math Functions
* ROUND / ROUNDUP / ROUNDDOWN
* ABS
* MOD
* SUMPRODUCT

### 7. Conditional Formatting
* Highlight duplicate values
* Color scales for performance analysis
* Data bars
* Top/Bottom Rules


## 🛠 Tools Used
* Microsoft Excel


## 🎯 Key Takeaways
After completing this section, the learner is able to:
- Understand core Excel formulas and functions
- Perform basic data calculations
- Clean and format data for better readability
- Apply logical thinking using Excel formulas
